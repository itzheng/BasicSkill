# CMake 介绍
~~~
CMake 是一个跨平台的安装（编译）工具，可以用简单的语句来描述所有平台的安装(编译过程)。

CMake 的配置文件取名为 CMakeLists.txt

CMake 并不直接建构出最终的软件，而是产生标准的建构档（如 Makefile）。

Makefile 定义了一系列的规则来指定哪些文件需要先编译，哪些文件需要后编译，哪些文件需要重新编译，甚至于进行更复杂的功能操作。
~~~
